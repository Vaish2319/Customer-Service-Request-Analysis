# Customer Service Request Analysis

## 📌 Description

This project analyzes NYC 311 customer service request data to identify complaint patterns, response time, and city-wise trends. The goal is to understand how different types of complaints are distributed and how efficiently they are resolved.

## 🛠 Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## 📊 Project Tasks

* Data loading and understanding the dataset
* Handling missing values
* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Visualization of complaints by city
* Analysis of response time

## 📈 Key Insights

* Identified major complaint types across NYC
* Analyzed complaint distribution by city
* Calculated and visualized average response time
* Observed patterns in service request handling
